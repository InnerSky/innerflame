# UI Components

This package contains shared UI components used across InnerFlame applications.

## Purpose
- Ensure consistent visual language across applications
- Avoid duplication of component code
- Provide a component library with proper documentation

## Components
- Base UI elements (buttons, inputs, modals)
- Composed components (forms, cards, layouts)
- AI-specific UI components (chat interface, suggestions)

## Usage
Components can be imported directly:

```typescript
import { Button, Input, AIResponseCard } from '@innerflame/ui-components';
```
