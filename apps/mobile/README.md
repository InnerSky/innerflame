# Mobile App (Planned)

This directory is reserved for the React Native mobile app implementation of InnerFlame.

## Planned Features
- Document viewing and editing
- AI assistance on the go
- Push notifications for document changes and AI suggestions
- Offline access to previously viewed documents

## Dependencies
- Will share core logic with the web application
- Will use shared types and UI components packages
- Will connect to the same backend services

## Timeline
- Planned for future implementation after core web functionality is stable

## Technical Considerations
- Will be implemented with React Native
- Planned to support both iOS and Android
