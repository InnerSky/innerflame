# API Server

This directory contains the REST API server for InnerFlame, handling data operations, authentication, and business logic.

## Key Components
- Express.js based REST API
- Supabase authentication integration
- Data access layer for entity management
- Middleware for request validation and auth

## Folder Structure
- `/src/controllers`: Request handlers for API endpoints
- `/src/middleware`: Authentication, validation, and other middleware
- `/src/services`: Business logic implementation
- `/src/routes`: API route definitions
- `/src/utils`: Utility functions and helpers
