# AI Service

This directory contains the WebSocket-based AI service for InnerFlame, providing real-time AI assistance for document creation and editing.

## Key Components
- WebSocket server for real-time communication
- AI agent implementation using LangGraph
- Firestore integration for session persistence
- Claude API integration for AI capabilities

## Folder Structure
- `/src/websocket`: WebSocket server implementation
- `/src/agents`: AI agent definitions and workflows
- `/src/models`: Data models for AI operations
- `/src/tools`: Agent tools for document operations
- `/src/types`: TypeScript type definitions
- `/src/utils`: Utility functions and helpers
